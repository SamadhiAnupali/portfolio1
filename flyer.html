Project Description - Admin Panel for Fit Sixes Cricket Fiesta Website

Role: UI/UX Design Team Member

Collaborated with the faculty to develop an admin panel for the Fit Sixes Cricket Fiesta website. Contributed to the UI/UX design process, focusing on enhancing user interaction and usability for the administrative backend of the website.

Key Responsibilities and Achievements

  • Contributed to wireframing and prototyping sessions for the admin dashboard.
  • Worked closely with developers to ensure seamless integration of design elements.
  • Implemented user-centric design principles to enhance the overall user experience for administrators.

Impact:

The admin panel design streamlined administrative tasks, improving efficiency for managing the cricket fiesta website, which received positive feedback from faculty members and organizers.
